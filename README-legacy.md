# DDL out! - 待办事项Deadline可视化系统 (v3.0)

## 项目简介
这是一个基于Python和WebView的现代化桌面待办事项管理系统。v3.0版本采用垂直堆叠布局，并将任务卡片重构为直观的进度条形式。

## 新增特性 (v3.0)
- **垂直布局**：分类按垂直方向堆叠排列，适应 9:20 的长宽比窗口。
- **进度条式任务卡片**：
  - 摒弃传统的文本列表，每个任务直接化身为进度条。
  - 进度条长度直观反映剩余时间比例。
  - 鼠标悬停显示精确的剩余百分比。
- **增强的分类管理**：
  - **折叠/展开**：点击分类标题前的箭头可折叠该分类，专注重要内容。
  - **局部新建**：每个分类标题栏右侧设有独立的"+"按钮，直接在该分类下创建任务。
  - **专业配色**：内置8种精选配色方案，同时支持系统原生取色器自定义颜色。
- **窗口锁定**：
  - 默认窗口比例锁定为 9:20 (450x1000)，保持紧凑、稳定的桌面窗口布局。

## 核心功能 (延续v2.0)
- **智能可视化**：根据紧迫程度自动调整颜色深度（0-12h最深 -> >72h最浅）。
- **拖拽管理**：支持直接拖拽任务在不同分类间移动。
- **自动排序**：剩余时间短的任务始终置顶。

## 技术栈
- **后端**：Python 3 + Flask + SQLite
- **前端**：HTML5 + CSS3 (Flexbox/Grid) + Vanilla JS (ES6+)
- **容器**：PyWebView

## 安装部署

### 环境要求
- Python 3.6+
- pip 包管理工具

### 安装步骤
1. 安装依赖包：
   ```bash
   pip install -r requirements.txt
   ```

### 启动项目
运行以下命令启动应用：
```bash
python app.py
```
应用将以 450x1000 的窗口大小启动。

## 开发说明
- 窗口尺寸在 `app.py` 中通过 `webview.create_window` 配置。
- 任务进度条样式位于 `static/css/style.css` 的 `.task-card` 和 `.task-progress-fill` 类中。
