# DDL out! 用户手册 (v3.1)

## 1. 界面概览
v3.1 延续了垂直堆叠布局，进一步优化了交互体验。
- **窗口**：固定为竖屏比例 (9:20)，系统会自动修正窗口大小以保持该比例。
- **滚动浏览**：当任务较多时，可以通过鼠标滚轮流畅地上下滚动查看所有分类。

## 2. 任务管理
### 进度条式任务
每个任务显示为进度条，长度代表紧迫程度，颜色代表剩余时间状态。

### 新建任务
1. **定位分类**：找到您想要添加任务的分类（如"工作"、"生活"）。
2. **点击按钮**：在分类标题栏右侧，点击宽大的 **"+ 添加待办事项"** 按钮。
3. **填写信息**：在弹窗中输入任务详情；截止时间的日、时、分可直接输入，也可使用下方滚轮选择，改变焦点后两者会自动同步。
4. **自动进位**：剩余时间中的分钟满 60 会进位到小时，小时满 24 会进位到天。
5. **保存任务**：确认分类与截止时间后点击保存。

### 移动任务
- 按住任务进度条，将其上下拖拽到其他分类区域即可。

## 3. 分类管理
### 折叠与展开
- 点击分类标题左侧的箭头，可以收起或展开该分类。

### 颜色设置
- 双击分类标题打开编辑窗口，支持预设色盘和自定义颜色。

## 4. 常见问题
- **Q: 为什么我调整窗口大小时它会自己弹回去？**
  - A: 为了保证最佳的视觉布局（9:20比例），系统会自动锁定窗口的长宽比。
- **Q: 任务太多显示不全怎么办？**
  - A: 直接使用鼠标滚轮即可上下滚动查看完整列表。
