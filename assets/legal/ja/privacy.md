# プライバシーポリシー

最終更新日：2026-07-15

DDL out!（以下「本アプリ」）は、ローカルファーストのデスクトップ・モバイルアプリです。本ポリシーは中国語版を解釈の基準とし、この翻訳は便宜のために提供されます。

## ローカルデータ

本アプリは、カテゴリー、タスク、締切、完了状態、バックアップ内容を開発者へアップロードしません。これらは端末内の SQLite データベースに保存されます。JSON バックアップは、利用者が選択したローカルファイルにのみ書き込まれます。

分析、広告、テレメトリーのサービスは組み込まれておらず、ユーザーアカウントやプロファイルも作成しません。

## 更新確認とネットワークアクセス

「起動時に更新を確認」は初期状態で有効です。「システムとデータ」で無効にできます。有効な場合、プロセス起動後に GitHub Releases API へ一度アクセスし、公開バージョン番号を比較します。「このアプリについて」から手動確認した場合も同様の通信が行われます。

更新リクエストにタスクやバックアップのデータは含まれません。ただし GitHub は通常の HTTPS 通信と同様に、IP アドレス、リクエスト時刻、`DDL-out update checker` User-Agent などの接続情報を受け取ります。処理には [GitHub のプライバシーに関する声明](https://docs.github.com/site-policy/privacy-policies/github-general-privacy-statement)が適用されます。

## 外部リンク

作者、ソースコード、コミュニティ、ダウンロード、文書へのリンクはシステムブラウザーで開きます。本アプリを離れた後は、移動先サイトのポリシーと規約が適用されます。

## 子どものプライバシー

本アプリは、子どもを含むいかなる利用者からも個人データを意図的に収集しません。

## 変更と連絡先

本ポリシーの更新はアプリのリリースとともに配布されます。公開可能な質問は [GitHub Issues](https://github.com/FlySparkle/DDL-out/issues) へお寄せください。公開 Issue に個人データ、データベース、バックアップを投稿しないでください。
