# 隐私政策

最后更新：2026-07-12

DDL out!（以下简称"本应用"）是一款本地优先的桌面与移动端应用。

## 数据收集

**本应用不收集、不上传、不分享任何个人数据。**

- 所有事项、分类、截止时间和完成状态均存储在您的设备本地 SQLite 数据库中。
- 本应用不接入任何分析、广告或遥测服务。
- 本应用不请求网络权限，不向任何服务器发送数据。

## 备份与导出

备份功能将数据导出为本地 JSON 文件，由您自行选择存储位置。本应用不会自动上传
备份文件至任何远程服务。

## 第三方服务

本应用不集成任何第三方 SDK 或服务。

## 儿童隐私

本应用不收集任何人的数据，儿童同样适用。

## 变更

本隐私政策如有更新，将通过应用更新渠道同步通知。
